# 本章概要：基础定义

本章主要讲解信号与系统的一些基础概念，包括信号的类型、能量/功率以及基本系统性质。

  讨论信号的 **能量 (Energy)** 和 **功率 (Power)** 概念。

## 2. 基本信号

- **冲激函数 (Impulse Function)**  

## 3. 系统的基本性质

1. **记忆性 (Memory)**  
   输出 $y(t)$ 不仅与当前输入 $x(t)$ 有关，还可能与过去或未来的输入有关。

2. **可逆性 (Invertibility)**  
   系统可以从输出恢复输入。

3. **因果性 (Causality)**  
   输出 $y(t)$ 只依赖于当前或过去的输入 $x(t)$。

4. **稳定性 (Stability)**  
   对有界输入，系统产生有界输出（BIBO 稳定）。

5. **时不变性 (Time-Invariance)**  
   如果输入信号延迟 $t_0$，输出信号也仅延迟 $t_0$。

6. **线性 (Linearity)**  
   系统满足叠加原理和齐次性。
