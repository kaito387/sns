# Chap 3 周期信号的傅里叶级数表示

## 1. 动机：从系统的“偏好”到信号的分解

在第二章中，我们用**卷积**求解系统输出。但卷积只是一种计算手段，它并没有告诉我们系统“长什么样”、有什么“偏好”。如果我们想洞察一个 LTI（线性时不变）系统的本质性质，该怎么办？

自然的想法是：**寻找这个系统的“特征向量”**。

考虑输入一个复指数信号 $e^{st}~(s\in \mathbb C)$。计算可知，在 LTI 系统下，其响应为：

$e^{st} \longrightarrow H(s)e^{st}$

其中，特征值 $H(s) = \int_{-\infty}^{+\infty} h(\tau)e^{-s\tau}\mathrm d\tau$。

系统对输入信号的频率和波形没有任何改变，仅仅是乘上了一个常数（复数，包含了缩放和旋转）。我们称 $e^{st}$ 为 LTI 系统的**特征函数**。

既然 LTI 系统对 $e^{st}$ 的处理如此简单（只需乘一个系数），那么**如果能把任意输入信号 $x(t)$ 分解为若干个 $e^{st}$ 的线性组合，求输出不就变成极其简单的乘法了吗？**

为了处理物理世界中常见的振荡信号，我们取 $s = j\omega$（纯虚数）。对于一个周期为 $T$ 的信号 $x(t)$，它的基波频率为 $\omega_0 = 2\pi/T$。我们希望将其按基波频率的整数倍展开，这就导出了**傅里叶级数（FS）**：

$x(t) = \sum_{k=-\infty}^{\infty} a_k e^{jk\omega_0t}$

## 2. 傅里叶级数的求解：正交空间的投影

怎么求系数 $a_k$ 呢？直接套用线性代数的直觉：

在复数函数空间中，我们可以定义两个函数的内积为它们在一个周期内的积分 $\langle f, g \rangle = \int_0^T f(t)g^*(t)\mathrm dt$。

奇妙的是，频率不同的复指数信号 $e^{jk\omega_0t}$ 彼此之间是**正交**的！它们构成了一组正交基。

要求 $x(t)$ 在某个基向量 $e^{jk\omega_0t}$ 上的坐标 $a_k$，直接做内积投影即可：

$a_k = \frac{1}{T}\int_{T} x(t)e^{-jk\omega_0t}\,\mathrm d t$

（除以 $T$ 是因为基向量 $e^{jk\omega_0 t}$ 自身的内积模长平方是 $T$，这就完成了归一化。）

**物理意义：**

傅里叶级数相当于把 $x(t)$ 这个**时间**上的信号，转换成了**频率**上的信号。横坐标是离散的频率点 $k\omega_0$，纵坐标是 $a_k$（复数，包含了该频率分量的振幅和相位）。

## 3. 数学严谨性与现实妥协

既然是无穷级数求和，就必然面临收敛性问题。能否保证所有周期信号都能这样展开？

答案是需要满足**狄里赫利条件（Dirichlet conditions）**：

1. 周期内绝对可积（保证系数 $a_k$ 存在且有限）。
2. 周期内极值点数目有限。
3. 周期内只有有限个第一类间断点。

**Gibbs 现象：**

即使满足了条件，在信号的不连续点处，无穷级数也会遇到麻烦。如果我们只取有限项（截断近似），在不连续点附近会出现无法消除的超量起伏（约 9%）。这暗示了：**突变（不连续点）意味着包含了极其丰富的无穷高频成分**，强行截断必然导致失真。

## 4. 傅里叶级数的性质（时频映射直觉）

记号：$x(t) \overset{\mathcal {FS}}{\longleftrightarrow} a_k$。

性质的推导不需要死记硬背，感受时域和频域的“联动”即可：

1. **线性：** 显然成立。

2. **时移：** $x(t - t_0) \overset{\mathcal {FS}}\longleftrightarrow a_k e^{-jk\omega_0t_0}$

   *直觉：* 时间上延迟了，形状没变（各频率成分的振幅 $|a_k|$ 不变），只是每个频率分量的起点变了（增加了一个与频率成正比的相位偏移）。

3. **时间反转：** $x(-t) \overset{\mathcal {FS}}\longleftrightarrow a_{-k}$ *(注：这里我修正了你的笔记，时间反转对应的是系数下标的反转)*

   *直觉：* 录音带倒放，相当于所有的相位都反过来了，正频率变成了负频率。

4. **时域尺度变换：** $x(\alpha t) \overset{\mathcal {FS}}\longleftrightarrow a_k$ 保持不变，但此时基频变成了 $\alpha\omega_0$。

   *直觉：* 把手风琴拉长或压缩，里面包含的“和声比例”没变（$a_k$ 不变），只是整体的音高（基频）变了。

5. **相乘：** $x(t)y(t) \overset{\mathcal {FS}}\longleftrightarrow \sum_l a_l b_{k-l}$ (离散卷积)

   *直觉：* 时域相乘，等于频域卷积。

6. **共轭与对称：** $x^*(t) \overset{\mathcal {FS}}\longleftrightarrow a_{-k}^*$ *(注：修正了你的笔记)*

   *推论：* 如果 $x(t)$ 是实信号，那么 $a_k = a_{-k}^*$（共轭对称，即幅度偶对称，相位奇对称）。如果实信号还是偶函数，那 $a_k$ 全是实偶数；如果是奇函数，那 $a_k$ 全是纯虚奇数。

**帕斯瓦尔定理（Parseval's Relation）：**

$\frac{1}{T}\int_T |x(t)|^2 \mathrm dt = \sum_{k=-\infty}^{\infty} |a_k|^2$

*直觉：* 能量守恒。在时域算总平均功率，等于把各个正交频率分量的功率直接加起来。正交基的好处就在于能量不耦合。

## 5. 离散时间的傅里叶级数 (DTFS)

（顺着前面的逻辑，我们现在进入离散世界。体会这里的“Aha moment”！）

考虑离散周期信号 $x[n]$，周期为 $N$。基波频率是 $\omega_0 = \frac{2\pi}{N}$。

我们依然想把它投影到离散的复指数基底 $e^{jk\omega_0 n}$ 上。

**核心质变（Implication）：**

在连续时间中，$k$ 从 $-\infty$ 到 $+\infty$，会产生无数个完全不同的极高频信号。

但在离散时间中，由于时间 $n$ 只能取整数，这导致了一个奇妙的现象：

$e^{j(k+N)\omega_0 n} = e^{jk\omega_0 n} \cdot e^{jN \frac{2\pi}{N} n} = e^{jk\omega_0 n} \cdot e^{j2\pi n} = e^{jk\omega_0 n}$

**发现了吗？在离散世界里，频率是具有周期性的！** 第 $k+N$ 个谐波，和第 $k$ 个谐波在离散采样下长得**一模一样**（这被称为频率混叠/Aliasing）。

**顺理成章的结论：**

既然只有 $N$ 个各不相同的复指数信号，那我们的傅里叶级数就不需要求无限项了！它**必然是一个有限项的求和**。这直接解决了连续时间中的所有痛点：

1. **没有收敛问题**：有限项相加绝对收敛，不需要什么狄里赫利条件。
2. **没有 Gibbs 现象**：用 $N$ 项就可以完美、精确地重建原信号，没有任何误差。

**DTFS 公式：**

$x[n] = \sum_{k=\langle N \rangle} a_k e^{jk\omega_0n} \quad \text{(综合方程)}$

$a_k = \frac{1}{N}\sum_{n=\langle N \rangle} x[n]e^{-jk\omega_0n} \quad \text{(分析方程)}$

*(注：$\langle N \rangle$ 表示在任意一个长度为 $N$ 的区间求和即可，因为所有东西都是 $N$ 周期的。)*

**DTFS 性质的特别注意点：**

由于基底只有 $N$ 个，这直接导致了**傅里叶系数 $a_k$ 也是一个周期为 $N$ 的序列**（即 $a_{k+N} = a_k$）。

这就形成了一个完美的对称：

**离散**且**周期**的时域信号 $x[n]$ $\longleftrightarrow$ **周期**且**离散**的频域信号 $a_k$。
