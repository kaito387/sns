# Chap 1 信号与系统基础

## 1. 动机：我们如何描述物理世界？

世界充满了物理量（信号）和处理这些物理量的黑盒（系统）。在深入计算之前，我们需要统一的语言来描述它们的大小、基本形态和行为法则。

## 2. 信号的“大小”：能量与功率

面对一个信号 $x(t)$ 或 $x[n]$，我们怎么衡量它有多“强”？

数学上，最自然的方法是算模长的平方积分（或求和），即**能量 (Energy)**：

$E = \int_{-\infty}^{\infty} |x(t)|^2 \,\mathrm d t$

*直觉与妥协*：如果信号是短暂的（比如一个脉冲），能量是有限的。但如果信号是无始无终的周期信号（比如正弦波），算出来的能量就是无穷大，这在工程上没法比较。

*顺理成章的引申*：既然总能量无穷大，我们就退而求其次，看看它“单位时间内的能量平均值”，这就是**功率 (Power)**。

所以：有限长信号看能量，无限长（周期）信号看功率。

## 3. 基本信号：构造万物的“乐高积木”

除了正弦波，我们还需要两种极端的信号：

- **单位跃迁函数 $u(t)$：** 描述一个“开关”动作。它天然适合用来把一个无始无终的信号“截断”成只有因果部分的信号。

- **冲激函数 $\delta(t)$：** 这不是一个普通的函数，而是一个“数学探测器”。

   *直觉*：它在时间上极其短暂，幅度无限大，但积分面积为 1。它的存在，是为了把连续的信号在某一个瞬间“提取”出来（采样特性）。

## 4. 系统的基本性质（测试黑盒的法则）

我们不知道黑盒（系统）里有什么，但可以通过观察输入输出，给黑盒贴标签：

1. **记忆性 (Memory)**：输出 $y(t)$ 是否只依赖于此时此刻的 $x(t)$？如果是，就是无记忆（比如简单的电阻）。如果依赖过去（甚至未来），就有记忆（比如电容需要积累电荷）。
2. **可逆性 (Invertibility)**：是否发生了信息丢失？如果不同的输入产生了相同的输出，信息就丢了，系统不可逆。如果能找到一个逆系统把 $x(t)$ 原样找回来，就是可逆的。
3. **因果性 (Causality)**：系统不能有“预知未来”的超能力。输出 $y(t)$ 只能依赖当前和过去的输入，不能依赖未来的输入（即系统不会在输入发生前就给出响应）。
4. **稳定性 (Stability)**：系统的输出不能“爆炸”。只要输入是有界限的（不飞向无穷），输出也必须是被限制在一个范围内的（BIBO 稳定）。
5. **时不变性 (Time-Invariance, TI)**：系统的物理属性不随时间改变。你今天测和明天测，只要输入延迟了 $t_0$，输出也会不多不少刚好延迟 $t_0$。
6. **线性 (Linearity, L)**：系统极其“乖巧”，满足齐次性（输入放大几倍，输出也放大几倍）和叠加性（两个信号一起输入，等于它们各自输出的加和）。
