# Chap 3：周期信号的傅里叶级数表示

第二章提到卷积可以求 LTI 系统输出，但卷积只能告诉你输出结果。如果想深入理解系统特性，就需要找出系统的“特征函数”。

---

## 1. LTI 系统的复指数特征

- 对复指数信号 \(e^{st}\) (\(s\in \mathbb{C}\))：
\[
e^{st} \longrightarrow H(s) e^{st} \quad \text{在 LTI 系统下仍是复指数}
\]
- 其中：
\[
H(s) = \int_{-\infty}^{+\infty} h(\tau) e^{-s\tau}\, d\tau
\]
- 这里，\(e^{st}\) 是系统的**特征函数**，\(H(s)\) 是对应的**特征值**。
- 类似线性代数，如果信号可以分解为特征函数的组合，系统对每个分量作用简单（乘上对应特征值）。

---

## 2. 周期信号的傅里叶级数

- 对周期 \(T\) 的信号 \(x(t)\)，基波频率 \(\omega_0 = 2\pi/T\)：
\[
x(t) = \sum_{k=-\infty}^{\infty} a_k e^{jk\omega_0 t}
\]
- 这就是 **傅里叶级数表示**，每个 \(e^{jk\omega_0 t}\) 都是特征函数。
- 系数 \(a_k\) 通过正交性求得：
\[
a_k = \frac{1}{T} \int_0^T x(t) e^{-jk\omega_0 t}\, dt
\]

> 除以 \(T\) 是归一化，让 \(a_k\) 表示真实幅度。

- 直觉上：傅里叶级数把时间信号 \(x(t)\) 转换到频率空间，横坐标是频率 \(k\omega_0\)，纵坐标 \(a_k\) 表示该频率成分的幅度和相位。

---

### 2.1 傅里叶级数存在条件（狄里赫利条件）

1. 周期内绝对可积  
2. 最大值和最小值出现次数有限  
3. 不连续点有限，且函数在不连续点上取有限值

- 不连续点附近截断级数会出现**Gibbs现象**：高频振荡和超量（幅度超过原函数）。

---

## 3. 傅里叶级数性质

\[
x(t) \overset{\mathcal{FS}}{\longleftrightarrow} a_k
\]

1. **线性**：\(\alpha x(t) + \beta y(t) \longleftrightarrow \alpha a_k + \beta b_k\)  
2. **时移**：\(x(t-t_0) \longleftrightarrow e^{-jk\omega_0 t_0} a_k\)  
3. **时间反转**：\(x(-t) \longleftrightarrow a_{-k}\)  
4. **时域尺度变换**：\(x(\alpha t)\) 的 \(a_k\) 不变，但对应的基函数变为 \(e^{jk\omega_0 \alpha t}\)  
5. **相乘**：时域相乘 ⇒ 频域卷积 \(a_k * b_k\)  
6. **共轭**：\(x^*(t) \longleftrightarrow a_{-k}^*\)  
   - 实偶函数 ⇒ 系数实偶  
   - 实奇函数 ⇒ 系数纯虚奇

---

## 4. 帕斯瓦尔定理

- 周期信号的平均功率 = 傅里叶系数模平方和：
\[
\frac{1}{T} \int_0^T |x(t)|^2 dt = \sum_{k=-\infty}^{\infty} |a_k|^2
\]

---

## 5. 离散周期信号

- 对周期为 \(N\) 的离散信号 \(x[n]\)：
\[
x[n] = \sum_{k=0}^{N-1} X[k] e^{j 2\pi k n / N}, \quad n = 0,1,\dots,N-1
\]
- 离散傅里叶系数：
\[
X[k] = \frac{1}{N} \sum_{n=0}^{N-1} x[n] e^{-j 2\pi k n / N}
\]

- 与连续情况类似：
  - 每个 \(e^{j 2\pi k n / N}\) 是离散特征函数  
  - 系数 \(X[k]\) 表示每个频率的幅度和相位  
  - 截断仍会有离散Gibbs现象
